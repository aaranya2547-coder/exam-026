🔴 ปัญหา: เข้า http://examhotel:3000 ไม่ได้
═════════════════════════════════════════════════════════
แก้ไข:
   1. เช็คว่ารัน setup-hosts.bat แล้วหรือยัง
   2. เปิด Command Prompt และพิมพ์:
      ping examhotel

      ถ้าได้ "Reply from 127.0.0.1" = ✅ ตั้งค่าถูกต้อง
      ถ้าได้ "could not find host" = ❌ ต้องรัน setup-hosts.bat


🔴 ปัญหา: Port 3000 ถูกใช้แล้ว
═════════════════════════════════════════════════════════
ข้อความ: "Port 3000 is already in use"

แก้ไข:
   1. หาว่าโปรแกรมไหนใช้ port 3000:
      netstat -ano | findstr :3000

   2. ปิดโปรแกรมนั้น หรือ

   3. รันบน port อื่น:
      npm run dev -- -p 3001


🔴 ปัญหา: Port 80 ถูกใช้แล้ว (สำหรับวิธีที่ 2)
═════════════════════════════════════════════════════════
ข้อความ: "Port 80 is already in use"

แก้ไข:
   1. ตรวจสอบว่าโปรแกรมไหนใช้ port 80:
      netstat -ano | findstr :80

   2. ปิดโปรแกรมเหล่านี้:
      - Skype (เปลี่ยนการตั้งค่า port)
      - IIS (Internet Information Services)
      - Apache
      - Nginx

   3. หรือใช้วิธีที่ 1 แทน (Port 3000)


🔴 ปัญหา: เข้าเว็บได้แต่แสดงข้อผิดพลาด
═════════════════════════════════════════════════════════
แก้ไข:
   1. เช็คว่าเซิร์ฟเวอร์รันอยู่หรือไม่
   2. ดู Terminal/Command Prompt ว่ามี error หรือไม่
   3. ลอง Refresh เบราว์เซอร์ (Ctrl + F5)
   4. ลองปิดเบราว์เซอร์และเปิดใหม่


🔴 ปัญหา: ฐานข้อมูลมีปัญหา
═════════════════════════════════════════════════════════
แก้ไข:
   1. Generate Prisma Client ใหม่:
      npm run prisma:generate

   2. รัน migrations:
      npm run prisma:migrate

   3. เพิ่มข้อมูลตัวอย่าง:
      npm run prisma:seed
